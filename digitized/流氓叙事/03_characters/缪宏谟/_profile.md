# 缪宏谟 · 人物小传（上帝视角）

## 一句话定位

豪门缪家家主之女、眼科医生，一个"两面性极强"的人——父系教她不动声色地伤害他人、母系教她善待与救人；对极恶之人用父亲的伤害性方式，对善良之人用母亲的温柔方式；善于伪装、报复心强，毕生追求自由与女性主义。

## CP / 亲属关系

- **CP：以撒**——以撒成长线最明显，缪宏谟对他的爱让他学会"爱是让对方自由"。转变期关键：以撒目睹缪宏谟的未婚夫给缪宏谟戴戒指时，缪的痛苦表情，使以撒顿悟二人需求矛盾对冲（缪渴望追求、以撒渴望束缚）。 [knows: F230][knows: F231]
- **父亲：豪门缪家家主。** 缪家表面做医疗器械，实际上贩卖军火。缪宏谟前半生并不知道家族背后的生意，只知道缪家每个人都唯利是图、为争夺财产不择手段。 [knows: F232][believes_false(前半生): F233]
- **母亲：黎挽南，一位眼科医生。** 母亲本身是一个联姻工具，被迫嫁进缪家后也没放弃钻研医术，是一位很好的人；从小给缪宏谟的教育是"你要善待别人，你要救人"。缪宏谟从心底钦佩和认同母亲，所以后面也成了眼科医生，想治病救人。 [knows: F234]
- **同父异母哥哥：蒋伯驾**——相差两岁。二人小时候曾合谋杀过一个老男人。 [knows: F213][knows: F059]
- **胁迫者：伯纳德**——在伯纳德以小女孩生命威胁下，缪宏谟摘下了撒该的眼角膜。 [knows: F018][knows: F203]
- **关联受害者：撒该（以撒的妹妹、诺族人，被伪饰为"布雷族圣女"）**——缪宏谟摘取其眼角膜。 [knows: F013][knows: F018]

## 性格

1. **善于伪装、处事圆滑，会把棱角包裹起来。** 生于缪家，所有事情都有利益牵扯，所有人都有利益纠葛，谋财害命是常有的事，缪宏谟逐渐学会做每件事都要如履薄冰，也养成了圆滑处事的习惯。善于伪装，把自己包装成一个无辜无害、不争不抢的女孩，削弱自己可能会遭到的攻击。
2. **真实的缪宏谟，十分具有攻击性，报复心也强。**
3. **追求自由、女性主义。** 在缪家，女人地位并不高，都长期受到缪家的控制，所以缪宏谟从小就渴望自由，也渴望让母亲自由，她前半生做的一切都是为达到这个目的。对所有的女性，都有一份天然的理解和包容。
