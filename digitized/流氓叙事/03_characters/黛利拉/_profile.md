# 黛利拉 · 人物小传（上帝视角）

## 一句话定位

布雷诺种族冲突中诞生的"战争之子"——从小缺爱、缺乏安全感，"追求爱的一生"；一个有温度的小杀手，活在以爱为唯一法则的小世界里，最终要在一次彻底的痛苦中学会自爱、接受失去与离别。

## CP / 亲属关系

- **CP：阿奇**——恋陪位。黛利拉演绎的终局是"阿奇消失"；设计上不设第二个结局，因为对黛利拉而言"没有，可能才是更好的结局"，要让她在彻底痛苦中学会自爱。 [knows: F250][knows: F251]
- **姐姐 + 妈妈：摩伊拉**——金狮医院大火中死了黛利拉的姐姐妈妈摩伊拉。黛利拉的姐姐妈妈被家暴并最终被害死，凶手阻止了黛利拉带她离开的计划。 [knows: F015][knows: F017]
- **原生母亲：** 黛利拉是"战争之子"——种族冲突中男人强暴了女性，生下的孩子被称为战争之子。这样的孩子一般得不到母爱，因为母亲会把自己受到的伤害和恨意转移到孩子身上，所以黛利拉从小就是缺爱的。 [knows: F252]
- **羁绊：奥丁**——陪她买最粉的裙子；曾让黛利拉"再当一分钟老大"（火柴燃烧的时间很短）。 [knows: F056]

> 命名提示：原文"姐姐妈妈摩伊拉"——指对黛利拉而言兼具姐姐与母亲身份的人，名摩伊拉；与"原生母亲（把恨意转移给她的母亲）"为不同对象，建图时区分。

## 性格

1. **缺爱、缺乏安全感。** 原生家庭有硬性（创伤），后来即使加入了新的家庭，也同样有被抛弃的经历，所以很强烈地需要很多爱，很多。
2. **在世俗的眼光下，黛利拉大概是"疯疯癫癫"的"精神病"，但其实她只是在做自己，活在自己的世界里。** 黛利拉当然也身处现实生活，但她其实是不在乎现实的很多东西的，比如道德、法律。她永远活在自己的小世界里，在这个小世界里，爱是唯一运行的法则。
3. **有温度的小杀手。** 黛利拉对于自己爱的人，是可以献出自己的生命和一切的。她虽然是一个杀手，应该是冷漠无情的人，但黛利拉其实很容易动感情，如果对方献出了百分之五十的爱，她会回报百分之一千。
4. **在别人看来，黛利拉情绪极度不稳定、行为夸张。** 在别人看来很小的事，但如果让黛利拉想了了不好的事情，她就会做出极其出格和夸张的反应。

## 设计初心 / 选角提示（手册原文要点）

> （3）关于黛利拉的角色设计初心：希望每一个黛利拉都能够学会**自爱**。

- 「大家看完黛利拉的人物剧本和演绎后，应该会有宝宝想问，为什么一定要阿奇消失？有没有第二个结局？可以有的，但对于黛利拉来说，没有，可能才是更好的结局。」
- 「黛利拉的一生中一直在给自己塑造了一个爱的堡垒，她想永远住在这个堡垒里面，但是她生活在一个现实的世界里，失去、离别不可避免地交替上演。我不希望黛利拉一直处于拒绝这份痛苦当中，我想她在一次彻底的痛苦中，学会自爱，学会自我满足，接受失去、离别，因为人的命运在有缺口之后，还会衍生出新的可能性。」
- 「希望每一个黛利拉，在最后的演绎里，看到镜子里的自己，和镜子上的 home，都能明白，自己、自己的爱才是最终的港湾。」
- 演绎执行提示：「千万不要忘记阿奇演绎最后一个步骤——在镜子上用红色水笔写上 home，摆好，在帘子拉开的时候，让黛利拉在镜子里看到自己。」
