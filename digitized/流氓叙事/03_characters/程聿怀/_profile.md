# 程聿怀 · 人物小传（上帝视角）

> 命名异写提示：本角色卡面统一作「程聿怀」。任务说明另列异写「程雨怀」，指同一人，建图时需归一。
> 另：本剧另有姓"程"的角色「程走柳」（程聿怀的亲妹妹），二者易混淆，务必区分。

## 一句话定位

一个出生于高知家庭、智商超群的「个人英雄主义者」——以 1994 年父母之死为分水岭，从温暖平和蜕变为漠然残忍；用一生执着追查父亲死亡真相，并为此一次次伤害最爱自己的人（羌青瓷），是全本「理解成本最高、事业线最重」的角色。

## CP / 亲属关系

- **CP：羌青瓷（又名 瑞法·莱诺 / 瑞法·莱诺=羌青瓷）**——主 d，恋陪位。两人 1990 年于东国大学（同学心理）相识相爱，2000 年结婚。 [knows: F006][knows: F010]
  - 命名异写归一：瑞法·莱诺 = 羌青瓷 = 程聿怀的爱人/妻或夫。
- **亲妹妹：程走柳**——比程聿怀小五岁，因理念不同于 1994 年后分道扬镳。 [knows: F200]
- **父亲：记者（1994 年前在世，1994 年遭污蔑、死亡——实为被莱诺家族害死于布雷诺）。** [knows: F201][believes_false 部分: 见下]
- **母亲：历史学教授（1994 年与父亲同年死亡）。** [knows: F201]
- **关键羁绊人物：阿奇**——1995 年起的好友/战友、君臣关系，二人共同组成布雷族军师符号「延迟」，各为二分之一的「延迟」。 [knows: F202]
- **敌对/操纵者：伯纳德·莱诺**（莱诺家族第一继承人，1995 年发现程聿怀查到真相，欲杀人灭口）。 [knows: F203]

## 男 / 女 版本差异（手册明确）

> 手册原文：「程聿怀人物本男女版本的差别主要集中于对羌青瓷的外貌、行为、性格的描述。女羌更温柔，男羌略强势。一般来说，为了主 d 更贴合文本里的描述，程聿怀的本需要根据羌青瓷的性别来定，女羌青瓷就发男程聿怀的本，男羌青瓷就发女程聿怀的本。」

- 差异**不在程聿怀本人的生平/性格**，而在其本中对 **cp 羌青瓷的描写**：
  - **女羌青瓷** → 配 **男程聿怀** 本；羌青瓷被描写得更温柔。
  - **男羌青瓷** → 配 **女程聿怀** 本；羌青瓷被描写得略强势。
- 即：发本规则 = 程聿怀与羌青瓷**性别反向匹配**（一男一女）。程聿怀自身小传两版一致。

## 性格

1. **极强的个人英雄主义。** 渴望像齐天大圣一样一人反抗整个权威——有野心、有目标、有勇气；但也自傲、自负，为了自己的目标会忽略剩下的一切，比如感情。
2. **对身边人有极强的保护欲，但并非真正的换位思考。** 当程聿怀把一个人划为"自己人"的范畴后，会极力保护，不管对方是否愿意接受，都会强加给对方；这个范畴之外的人，都不在其考虑范围之内。
3. **内心傲慢，因此表现为对其他人的漠视、残忍。** 这份傲慢一是来源于本身能力的卓越；二是出生于高知家庭，家庭教育让 TA 从小看待问题辩证而理性，但也因此缺少了情感和感性。
4. **无情、凉薄是创伤性的。** （对照蒋伯驾「题外话」：在理性、辩证看待事情和做决定上，蒋伯驾与程聿怀很相似，但蒋伯驾对生命谦逊尊重，程聿怀更加无情、凉薄。）程聿怀几乎不会考虑生命本身意味着什么，TA 只做对的事，很少考虑代价。父母死亡、和郭松（〔?〕，原文"郭松"，疑为另一人物或异写）意气争吵后，已撕裂了对世界保有的最后一份期待和天真，自此生出一层无情的保护壳，很少有情感能够走进 TA 的心里；作出决定后，TA 会刻意屏蔽掉决定代表的牺牲和牺牲背后的个体的伤痛。

## 设计初心 / 选角提示（手册原文要点）

- 「程聿怀的理解成本、信息量真的很高，事业线比较多。如果玩家是冲着谈恋爱来的，不太想搞事业，程聿怀真的、真的很不合适！」
- 「这个位置跟市面上的恋陪位诉求是有一点相悖的，TA 很大篇幅都在搞事业……大部分就是在干巴搞事业，很干巴！非常、非常、非常不建议玩传统恋陪位的玩家宝宝们盲锁！」
- 「反之，平常不爱玩恋陪位的事业脑们，可以尝试一下程聿怀。」
- 提问玩家以判断适配：「是否执着地追求个人理想？如果是的话，是否可以因此放弃爱情、亲情？如果两个答案都是『是』，其实可以考虑程聿怀。但如果第二个回答是『都行、可以吧』这种明明可以放弃还要装作在乎的死样，那就真的很贴程聿怀了。」
