## 五、目录结构说明

数字化产物根目录：`digitized/流氓叙事/`，按内容职能分区编号：

| 目录 | 内容 |
|---|---|
| `00_index/` | 索引区：`manifest.md`（本文件，物料清单）、`entities.yaml`（人物名册/异写对齐）、`timeline.md`（上帝视角时间线） |
| `01_world/` | 世界观设定（莱诺家族 / 诺族 / 布雷族 / 狂草帮等背景） |
| `02_truth/` | 真相还原与阴谋网（DM 上帝视角，含事实 ID F1xx/F2xx/F7xx） |
| `03_characters/` | 6 名 PC 人物小传（以撒 / 程聿怀 / 缪宏谟 / 程走柳 / 蒋伯驾 / 黛利拉），每人 `_profile.md` |
| `04_npcs/` | 3 名 DM 扮演 NPC（羌青瓷 / 奥丁 / 阿奇），含 `_profile.md` / 演绎脚本 / 道具与 BGM |
| `05_acts/` | 分幕 DM 主持流程脚本（序幕 / 第一～七幕 / 死亡演绎 / 结局演绎剧场） |
| `06_clues/` | 线索卡数字化（按幕分子目录）+ `clues.csv`（线索总表）+ `_谜面清单.csv`（待解/已解谜面登记） |
| `07_assets/` | 资产区：勘误、心测、音频索引（`audio/_audio_index.md`） |
| `_ocr/` | OCR 中间产物 |
