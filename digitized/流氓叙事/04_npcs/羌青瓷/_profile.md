# 羌青瓷（瑞法·莱诺）人物档案

> 命名异写说明：本NPC在剧本中有两个名字——
> - **瑞法·莱诺（Rifa Lainuo）**：本名，莱诺家族成员（西国）。
> - **羌青瓷**：瑞法后来自取的东国名字。
> 二者为同一人。备注：手册正文使用「TA」指代羌青瓷，演绎台词中允许「他/她」「学弟/学妹」「学姐/学长」按演绎者性别切换，说明本角色可男可女演绎。
>
> 其他相关命名异写：**程聿怀（女主/男主之爱人）** 与 **程走柳（程聿怀的亲妹妹/亲弟弟）** 是两个不同人物，手册中两名字反复出现，请勿混淆；本手册个别处把「程聿怀」误排为「程走柳」，已在脚本中按上下文标注。**蒋伯驾**（手册原文用字，疑同「将伯驾/蒋伯驾」体系）。**伯纳德·莱诺** = 莱诺家族第一继承人、羌青瓷兄长。**夏尔·莱诺** = 羌青瓷父亲。**黎梦鲸** = 羌青瓷母亲。

## 一、生平（编年）

[reveals: F001] 莱诺家族是百年新闻世家，享誉国际，创立了世界公认最高新闻奖【莱诺新闻奖】。
[reveals: F002] 但莱诺家族实为道貌岸然——经常靠煽动舆论、挑起战争，并借机售卖军火，大发战争横财。

| 19世纪中期 | 莱诺家族兴起，成为百年新闻世家 | F001 |
| —— | 父亲夏尔·莱诺在心理治疗过程中对患者（东国心理医生黎梦鲸）产生移情，错误地将患者对医生的依赖当作爱情 | F003 |
| —— | 夏尔凭莱诺家族势力强娶黎梦鲸，并将她囚禁身边，使其心理出现问题、行为极端 | F003 |
| **1970年** | 瑞法·莱诺出生于西国莱诺家族，为黎梦鲸在囚禁期间所生（囚禁、强制的恶果） | F004 |

| **1995年** | 程聿怀为寻真相来到布雷诺，从瑞法电脑中发现真相，却被莱诺家族第一继承人伯纳德·莱诺发现端倪。伯纳德欲杀程聿怀灭口，瑞法不允许，与他做交易：瑞法以后为他做事，并用火柴疗法（一种催眠手段）消除程聿怀记忆，换取程聿怀平安 | F008 |

| **2000年** | 莱诺家族想在布雷诺重新挑起战火，需与「延迟」合作乃至联姻；瑞法主动申请，获家族同意，再次来到布雷诺。TA给自己取东国名字——羌青瓷 | F012 |
| 2000年 | **取名缘由**：因程聿怀为追求真相，在脸上专门纹了青瓷提示自己永远要追求清白，瑞法把"青瓷"变成TA的名字。又据程聿怀讲过的孤挺花传说中"总有一个牧羊人"，东国牧羊人祖先可溯至古代羌族（羌姓，蚩尤一支，西戎牧羊人，天生喜欢细雨咽咽的草原），故"羌"成为TA的姓氏。自此"羌青瓷因程聿怀而生" | F013 |
| 2000年 | 二人结婚。此后五年羌青瓷演出"不爱程聿怀"的样子，让程聿怀以为TA只是为利益联姻；实际上羌青瓷痛苦地压抑着自己所有的爱 | F014 |
| 2000年后 | 程聿怀需在众人面前编造相爱故事，羌青瓷提议就说他们是恩爱多年的大学情侣。程聿怀只觉羌青瓷擅长说谎，后来发现羌青瓷说的所有话全部都是真的 | F015 |
| **2005年** | 在羌青瓷帮助下，程聿怀和其他人一起揭穿莱诺家族真面目；程聿怀也狠心让人把羌青瓷心中的芯片取出，羌青瓷因此死亡 | F016 |
| 2005年（死后） | 程聿怀才知道原来他们从1990年就相爱了。羌青瓷爱了程聿怀十五年，却沉默了十年 | F017 |

> 剧本现实时间线（演绎发生于）：**千禧年（2000年）金狮医院开业当天**，羌青瓷任金狮医院院长。剧本内羌青瓷正在执行一个筹谋已久的计划（见下"阵营定位"）。

## 二、性格（手册标注"需要转换"）

1. **前期**：看起来优雅端庄，对大家十分友好和善。
2. **第四幕以后**：开始在程聿怀面前演坏人，需要阴阳怪气、激起所有人的仇恨。
3. **进入羌青瓷的两个演绎后**：状态回归正常的温柔即可，就像卸下了面具一样轻松，但带着一点遗憾。

> 核心矛盾：一个被训练成"不懂爱、只懂什么不是爱"的人，最终无可救药地爱上了治愈TA的程聿怀；用一生说谎来保护爱人，自己却先一步赴死。"很擅长说谎"，但"说的所有话全部都是真的"。

## 三、形象 / 服装 / 道具

**外在特征：**
1. 胸前佩戴一朵红色的孤挺花（孤挺花花语：爱要勇敢地追寻——此意象源自程聿怀，程聿怀胸前总戴一朵炙红色孤挺花）。
2. 前期会穿白大褂（院长身份），在第四幕演绎时需要脱掉。
3. 喜欢穿绿色，女生可以佩戴珠宝。
4. 男生其实可以外面白大褂，里面白色衬衫，佩戴一个绿领带，不需要很大面积的绿色。

**着装执行细节（互动1准备）：**
- 羌青瓷里面穿一件绿色裙子（男羌西装或者衬衫），外面穿一件白大褂，把扣子扣上，先不要漏出里面的裙子或者西装/衬衫。
- 第四幕出场演绎结束→脱白大褂→露出礼服/西装，自报"瑞法·莱诺"。

**贯穿道具：**红色孤挺花（胸口）、白大褂、绿色服饰；高脚杯、白葡萄酒（rio代替）、吐真剂（茅根竹蔗水/什么水都行）、七份病例线索/诊疗书、录音笔、黑色羽毛（莱诺家族标志渡鸦）、鲨鱼手环、镜子、一盒火柴、椅子、手铐。

## 四、阵营定位

- **真实身份**：莱诺家族成员瑞法·莱诺；剧本时点已是/即将成为**莱诺家族家主**。剧本现实身份为**金狮医院院长**。
- **表面立场（第四幕后）**："我将代表莱诺家族，站在程聿怀，站在你，站在你们所有人的对立面！"——主动充当所有玩家（复仇者们）的"反派靶子"。
- **真实立场**：这是一场**自导自演、筹谋已久的"局"**。羌青瓷暗中帮助复仇者们（怒河/狂草帮等）聚集力量、解开隔阂，提供关键证据（录音笔），并安排奥丁来"救"被绑的玩家。目的：替程聿怀和程走柳还其父清白，推翻莱诺家族；自己甘当被复仇、被剖心而死的"祭品"。
  - [reveals: F011] 莱诺家族致命把柄=植入历代家主心脏的芯片。
  - [reveals: F020] 羌青瓷在第五幕坦白时对程走柳挑明："伯纳德是你杀的，怒河的创建人也是你。"——即**杀伯纳德者、怒河创建者均为程走柳一方（复仇者），不是羌青瓷**。羌青瓷只是知情并据此判断怒河力量不足、需聚集各方。（DI 原文页21-22 / _ocr 行205）
  - [reveals: F018] 羌青瓷一直保留着1994莱诺新闻奖颁奖台上程聿怀父亲临终20秒遗言的录音（伯纳德命人销毁了音频，但羌青瓷私留一份），内含程聿怀父亲遗言、劳伦、舅舅肖须言生前录音。

## 五、与玩家的羁绊

### 程聿怀（核心CP，"恋爱角色"）
- 手册特别强调："饰演羌青瓷的DM需要认真阅读所有玩家剧本，特别是程聿怀的剧本。作为恋爱角色来说，非常重要。"
- 1990相识相爱→1995被羌青瓷洗去记忆→2000假意联姻、互相说狠话演恩爱→2005程聿怀取芯片致羌青瓷死。
- **互动模式**：前期当众喊"老公""夫人"，营造甜蜜假象；私下在程聿怀耳边说狠话（如倒水后说"你真喝呀～我下毒喽～"）。中期程聿怀会觉得羌青瓷恶心、会骂羌青瓷，此时羌青瓷笑着阴阳怪气地回去，适度气一气程聿怀。
- 终极保密：羌青瓷洗去程聿怀记忆的真实原因（保护程聿怀+自觉配不上），以及"我和ta曾经真的相爱过"——**绝不能让程聿怀知道**。"就让程聿怀继续恨我吧，只有恨我，ta才能得到自己想要的一切！"

### 程走柳（程聿怀的亲妹妹/亲弟弟）
- 羌青瓷与程走柳有最多次私聊（互动2、6、7，共3次）。
- 羌青瓷以"心理医生"身份告知程走柳：程聿怀因1995布雷诺调查受刺激患"创伤失忆症"，请程走柳保守秘密、配合假装；承诺"我一定会治好你哥哥/姐姐的"。
- 第五幕坦白：程聿怀的记忆是羌青瓷亲手洗去的；交付录音笔，约定5月2日仪式上当羌青瓷问"你们有人证吗？"时，程走柳拿出录音说"我有！"。
- [knows: F020] 同幕羌青瓷向程走柳点破"伯纳德是你杀的，怒河的创建人也是你"——即程走柳即为杀伯纳德者、怒河创建方；羌青瓷据此判断怒河势单、需聚各方力量正面对抗莱诺。
- [knows: F009][knows: F018] 程走柳是唯一被羌青瓷托付真相与证据的玩家。

### 蒋伯驾
- 仅1次私聊（互动5）。蒋伯驾知道卢特筹划了两年大屠杀，想阻止，找羌青瓷配合（因看出羌青瓷与伯纳德不同、会在乎人的性命）——【受控燃烧】计划。
- 羌青瓷回应：会考虑发布广播邀请诺族官员来金狮医院免费体检（暗示以体检为名行控制之事，"以火灭火"）。

### 黛利拉（Delilah）
- 第四幕病例演绎中为"一号病人"，病症：躁郁？嗜血？病因不明。自述否认有病，只想把发起屠杀的混蛋撕碎；更恨那个**家暴并害死了她"姐姐妈妈"的混蛋**——那人阻止她带"姐姐妈妈"离开的计划、想把她关起来，最终害死了"姐姐妈妈"。黛利拉誓言"我一定会找到他，**一寸一寸撕开他的喉咙**"——这正是她"匕首"动机的来源。（DI _ocr 行248-251）
- 第四幕奥丁对黛利拉："黛利拉，拿好匕首，随时准备割开他们的喉咙。"羌青瓷对黛利拉："黛利拉，失去爱痛苦吗？记住这种感受，别压抑你的痛苦。"——羌青瓷在第四幕意在**激起每位玩家的仇恨**，把他们拧成对抗莱诺家族的合力。

### 奥丁（Odin）
- 关键"搭档/对手戏"对象。奥丁是大屠杀亲历者/见证者/复仇者（非"幸存者"）。第四幕病例演绎压轴登场，与羌青瓷有长段对手戏；最终二人达成默契（"狂草帮没有孬种""所以。"），由奥丁负责后续"救"被绑玩家。

### 其他玩家（病例演绎，第四幕：各病人自述要点）
- 二号病人**以撒**：表面"狂笑症"，自称真正的是"说谎者'恐惧'症"。其妹妹**撒该**本是诺族小女孩，被人用"一张轻飘飘的照片"造谣、变成布雷族的"圣女"；以撒的祖母为了让刽子手相信以撒也是参与屠杀的布雷族人，**死在了以撒手上**（以撒为自保亲手杀祖母）。自称"**逃跑者以撒**"。（DI _ocr 行254-260）
- 三号病人**缪宏谟**：恐惧症，恐惧物品=手术刀。被伯纳德"那个魔鬼"**用那些小女孩的生命威胁，被迫亲手摘下撒该的眼角膜**——由此串联 缪宏谟—伯纳德—撒该 三方，缪宏谟自述"此生我都会活在罪孽当中"。（DI _ocr 行264-267）
- 四号病人**程走柳**：恐惧症，恐惧物品=相机、新闻。自述想抛弃镜头（莱诺家族让它变得肮脏），"在伯纳德的逼迫下，我与劳伦一样，成了新闻的罪人"。（DI _ocr 行269-275）
- 五号病人**蒋伯驾**：病症不清楚、在压抑自己，病因不可告人。自述"我救人，也杀了人……死去的人足以让伯纳德很满意"，自问走到最后还救不救得了。（DI _ocr 行278-283）
- 六号病人**程聿怀**：欺瞒自己的爱人，病因=野心？不坦然？欲望。以"记者只阐述事实，无可奉告"回避；末句对羌青瓷说"等我找到真相的时刻，你会与我一同沉沦的是吗？爱人。"羌青瓷答"我会比你先一步沉沦其中。"（DI _ocr 行285-288）
- 压轴**奥丁**：自报"奥丁"。
