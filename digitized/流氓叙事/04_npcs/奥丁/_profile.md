# 奥丁 · 人物档案

> 本档案据《流氓叙事》奥丁演绎手册"奥丁生平 / 性格特点 / 形象"部分整理。叙事正文逐字保留，未概括改写。

## 一、命名与异写说明

- **奥丁**：本NPC，狂草帮、怒河老大。
- **蒋伯驾**：手册原文写作"蒋伯驾"（注意：本项目命名异写表中作"将伯驾/蒋伯驾"，此处统一以原文"蒋伯驾"为准）。1995 年东国负责兴建电台的负责人，后为奥丁所喜爱、视为家人。
- **羌青瓷**：原文"【羌青瓷】（又名：瑞法·莱诺）"——即羌青瓷 = 瑞法·莱诺，金狮医院院长，莱诺家族成员，奥丁的对立面。（命名异写表中亦记此等同关系。）
- **程聿怀 / 程走柳**：原文同时出现"记者【程走柳】"与"记者【程聿怀】"两人，并非异写，是两个不同角色。下文"怒河"创始人为"程走柳"与"缪宏谟"。性格特点条目中又出现"把程聿怀当作同行者"。请注意区分：**程走柳（怒河幕后创始人之一）** 与 **程聿怀（被当作同行者的记者）** 在本手册中是分列的两个名字。（备注：本项目命名异写表列有"程聿怀/程雨怀"，但本手册未出现"程雨怀"写法。）
- **缪宏谟**：金狮医院副院长；怒河幕后真正创始人之一。
- 涉及地名/族群：【布雷诺】（国）、【布雷族】、【诺族】；【莱诺家族】（西国）、【缪家】（东国）。
- 涉及组织：【狂草帮】、【怒河】；机构：电台（1995）、【金狮医院】（2000）。

## 二、生平（编年）

> 以下为手册"奥丁生平"正文逐字转录，分条编年。

- **1960 年**：奥丁出生于【布雷诺】。[F001]
  - 【布雷诺】是世界上最穷苦的国家，居住在这里的两大族人——【布雷族】和【诺族】，百年来互相仇恨、对立。种族之间的暴乱无限循环，从未得到过真正的平息。[F002]
  - 奥丁的前半生经历了无数次种族间的战争。
  - 他出生在一个拥挤的家庭，兄弟姐妹众多，生活如同一场无休止的喧闹。
  - 在这喧闹中，他找到了自己的避风港——耳机。耳机连着收音机，里面总会放着几首邻国音乐频道的摇滚乐曲。摇滚乐成了他与世界隔绝的屏障，让他在嘈杂中寻得片刻的宁静。[F003]

- **奥丁十二岁那年（约 1972）**：种族战争再次爆发，他和家人们一起逃命，但是他的兄弟姐妹们一个接一个地倒下，而他则在他们的保护下，艰难地活了下来。[F004]
  - 家人的牺牲让他刻骨铭心，那一声"跑啊，奥丁！"如同一道命令，让他在绝望中找到了前进的方向，让他在接下来的人生中，总是跑步前进。
  - 并且，奥丁从此经常听《born to run》（生来奔跑）。[F005]
  - 失去家人的奥丁，没有沉沦，而是站了出来，组建了【狂草帮】。[F006]
  - 他把因为战争而无家可归的孤儿、残疾人、老人全部收容进了他找到的一个荒废基地，并跟大家一起把这里经营成了一个大家庭。他也成为了这个新家庭的领导者，用摇滚乐和笑话驱散每个人心中的阴霾，尽管他的笑话并不好笑，但他从未放弃。

- **狂草帮性质**：狂草帮同时收纳两个种族的人，在这里不分没有种族之分，大家和平共处。[F007]
  - 这一点也成为布雷诺其他人讨厌狂草帮的原因，因为他们认为狂草帮忘记了种族百年仇恨和历史。
  - 奥丁教授狂草帮很多人如何使用武器、如何保护自己，后来为了谋生，狂草帮很多人成为了打手、杀手。
  - 奥丁本人也很憎恨在战争中施暴的凶手，所以如果接到的杀人订单，正好是屠杀中的刽子手，他就会主动为对方打折。[F008]
  - 因为奥丁爱好和平、讨厌挑事的人，所以，狂草帮也会在奥丁的带领下去维护些社会治安，算半个义警。

- **一九九五年（1995）**：西国的【莱诺家族】和东国的【缪家】来布雷诺兴建电台。[F009]
  - 他们的表面目的是做公益，给布雷诺支援公共设施，实际目的让布雷族和诺族争这个电台，挑起战争，借机卖武器，发战争财。[F010]
  - 但奥丁并不知道这个实际原因，他只看到有外国佬来布雷诺，建了一个电台，然后种族战争就开始了。[knows: F009] [误信/不知: F010]
  - 自此以后，奥丁就非常讨厌外国人，更讨厌外国人来布雷诺建公共设施。
  - 那段时间，奥丁知道那个负责兴建电台的东国负责人叫做【蒋伯驾】，他深深记住了这个名字，并很厌烦这个人。[F011]

- **一九九六年（1996）**：十五岁的黛利拉凭借出色的身手加入了狂草帮。[F012]
  - 她成为了一个小打手，接活赚钱，养自己的家人。
  - 后来奥丁知道黛利拉是战争之子，也就是在大屠杀中男人强暴女人生下的孩子，身世可怜，奥丁很心疼这个女孩，开始把她当女儿一样疼爱。[F013]
  - 五大三粗的他开始摸索如何养一个女儿，他开始问很多人的意见只为了给黛利拉去挑一个好看的裙子，用自己的大斧头去给黛利拉的裙子裁线头。
  - 奥丁因为黛利拉变得细腻、多了耐心，不厌其烦地想用亲情让这个天天嚷嚷着要许多爱的小女孩感觉到足够的安全感。

- **同年（1996），没过多久**：奥丁又认识了以撒。[F014]
  - 布雷诺有一个规定，是这里的人不可以慢跑，因为急促的脚步声被认为是激化冲突的事情。但奥丁从不屑于遵守没有道理的规矩，所以当他看到明明很想跑、却不敢跑的以撒时，他上去抓着以撒跑了起来。
  - 以撒问他，布雷诺不是不能慢跑吗？奥丁说，那只要跑得够快，就可以！
  - 那天以撒戴上了奥丁的耳机，听到里面摇滚的轰鸣，那首歌曲，就是《born to run》。那天之后以撒已经加入了狂草帮。
  - 奥丁觉得跟以撒有很多共同点，比如说喜欢跑步、喜欢听 born to run，最重要的是，以撒有狂笑症，他有的时候会控制不住地大笑，而奥丁的笑话总是没人捧场，但以撒会笑，于是奥丁开始不厌其烦地给以撒讲笑话。[F015]
  - 奥丁有一个笑话小本，专门记预备讲给以撒听的笑话，并会在上面批注以撒听到这些笑话时的反应。[F016]
  - 布雷诺很多人不知道狂笑症是什么东西，都觉得以撒是邪灵入侵了，说他有病，但奥丁觉得以撒很正常，一个人想笑就笑，算什么疾病？奥丁觉得说以撒不正常的人才是脑子有病。

- **2000 年**：西国的【莱诺家族】和东国的【缪家】故技重施，又要在布雷诺新建【金狮医院】，以此挑起更大的战争，谋取更多战争财。[F017]
  - 奥丁觉得又是外国佬建东西，肯定会引起危机，于是带着狂草帮的人在金狮医院开业仪式上大闹一场，想要把这些外国佬都赶回去。
  - 也是在这次闹剧中，他认识了【金狮医院】的院长【羌青瓷】（又名：瑞法·莱诺）、副院长【缪宏谟】、记者【程走柳】、记者【程聿怀】，最后，还见到了 1995 年那个建电台的蒋伯驾。[F018]
  - 大家不打不相识，在这次之后，奥丁发现这几个人品性还不错，但也并不是完全地相信他们，直到后来，金狮医院发生小规模械斗，这几个人和狂草帮一起共同对抗暴民，维持治安，奥丁才真正地相信他们。
  - 也是这一次，奥丁对蒋伯驾彻底改观，他发现这个小子身手不错也仗义，并不像自己想象中是一个很冷漠不负责任的上位者，奥丁开始喜欢蒋伯驾。[F019]

- **2000 年中旬**：【莱诺家族】和【缪家】共同挑起种族大屠杀开始了。[F020]
  - 布雷族疯狂地屠杀诺族人，在这场屠杀中，狂草帮的人为了保护平民、维护治安几乎都死去了。[F021]
  - 大屠杀结束后，奥丁察觉出莱诺家族在其中一定脱不了干系，并在这个时候收到了政府的指令——所有经历大屠杀之后的人必须进行心理治疗。
  - 而给他治疗的人正是莱诺家族的人——羌青瓷，在这一次治疗中，羌青瓷暗示奥丁大屠杀就是莱诺家族挑起的，挑衅奥丁，向他们家族复仇。[F022]
  - 从此，代表莱诺家族的羌青瓷和奥丁彻底站在了对立面。[knows: F022]

- **约 2001 年（"一年之后"）**：程走柳和缪宏谟找到了奥丁，说他们要向莱诺家族复仇，并且她们提供军火和钱财，准备组织一个正规的复仇组织。这个组织，叫做【怒河】。——愤怒竟然无处发泄，那就汇聚成河。[F023]
  - 这个组织的目标就是杀死 2000 年大屠杀幕后的所有操纵者，包括莱诺家族，奥丁同意了下来，并成为了怒河表面上的指挥官。[F024]
  - 而程走柳和缪宏谟这两位怒河真正的创始人，因为不想暴露身份，一直隐居幕后。[F025]
  - 她们两个在大屠杀中被莱诺家族威逼着害死了无辜的人，觉得自己罪孽深重，想要在怒河杀光所有凶手之后，把她们两个也杀了。[F026]
  - 但奥丁知道她们的苦处和难处，不觉得她们该死，开始想方设法劝她们活下来，但两人死志坚定，奥丁只能暂时答应她们。奥丁说，她们的死法只能由奥丁来决定。但实际上奥丁给她们想的死法只是自然老死。[F027] [knows: F026]

- **后来**：在一次机缘巧合下，羌青瓷绑架了主角所有人，并在他们之间挑拨离间，还好奥丁及时赶到，把所有人救下，并且带领所有人明确了一起向莱诺家族的复仇目标。[F028]

- **2005 年**：莱诺家族虚伪地举办大屠杀 5 周年纪念活动的时候，主角所有人在会场上曝光了莱诺家族所有的罪行。[F029]
  - 而这个时候，奥丁就替他们在他们的大门口替他们守着，不让任何一个莱诺家族的人进来打扰他们。
  - 也是在这场战役里，奥丁和怒河所有人都牺牲了，但也成功守住了大门。[F030]
  - 奥丁的一生永远戴着耳机，他用耳机隔绝了很多暴力和混乱的声音，但是在他生命的最后，他那个播放着《生来奔跑》的耳机也被子弹打穿了，里面再没有音乐，他踏上了一条无声之旅。[F031]

## 三、性格特点（手册原文）

1. 幽默搞笑，爱讲冷笑话，开朗话唠，不让话掉在地上。
2. 乐观、单纯、不聪明，经常进入聪明人（比如羌青瓷）的语言陷阱，一听好话就很容易上当。
3. 重视亲情，人际关系中最高待遇是把一个人当"家人"，把黛利拉当女儿，把以撒当儿子，把缪宏谟和程走柳当朋友，想要让蒋伯驾也做自己的家人，觉得蒋伯驾如果出生在布雷诺，一定会进狂草帮，把程聿怀当作同行者。
4. 重视责任担当，有领导力和保护欲。作为狂草帮和怒河的老大，想要保护里面的每个人，如果有人受到伤害，会非常沮丧。

## 五、阵营定位

- **早期**：【狂草帮】创始人兼老大——跨种族收容所/义警性质的武装团体，反对外国势力（莱诺家族、缪家）介入布雷诺。
- **中后期**：【怒河】表面上的指挥官（名义最高指挥），实际幕后创始人是程走柳与缪宏谟。目标：复仇莱诺家族 / 诛杀 2000 年大屠杀所有操纵者。
- **对立面**：莱诺家族，及其代表羌青瓷（瑞法·莱诺）。

## 七、关键保密信息（复盘才揭示，演绎中暗藏）

- 奥丁明知程走柳、缪宏谟把自己写进怒河暗杀名单、抱有死志，他"帮她们挑好了死法"实则是"自然老死"——这是善意的瞒骗。[F027]（死亡演绎中以"我可没有骗她们……活到自然死亡也算是死法"点出。）
- 奥丁不知道电台/医院的"实际目的"是莱诺家族与缪家发战争财——他只是凭直觉讨厌外国人建设施。[F010 vs knows:F009]
- 奥丁一生在流程里提的歌、沉浸的歌是《Don't Cry》（非《Born To Run》）；《Born To Run》是生平叙事用歌，不适配文本与流程。详见 道具与BGM.md 之 PS 说明。
