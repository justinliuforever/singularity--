## 一、阵营总览

- **加害方·西国莱诺家族**：家主瑞法·莱诺（=羌青瓷）；第一继承人/羌青瓷兄长 **伯纳德·莱诺**；羌青瓷叔叔 **卢特·莱诺**（年近六十，娶程走柳、收蒋伯驾为义子）；家族祖辈夏尔·莱诺。动机：靠新闻煽动战争、贩卖军火、发战争横财。 [F101][F103][F106]
- **加害方·东国缪家**：缪家家主（缪宏谟与蒋伯驾之父）。表面医疗器械，实则军火，是莱诺家族的军火合伙人。 [F103][F140][F151]
- **复仇方·怒河（复仇组织）**：真正创始人=**程走柳 + 缪宏谟**（隐居幕后）；表面指挥官=**奥丁**；杀手=以撒、黛利拉。资金/军火由程走柳、缪宏谟提供。目标：杀死2000年大屠杀所有操纵者（含莱诺家族）。 [F130][F141][F182]
- **复仇方·布雷族领导团体（"延迟"）**：程聿怀（半个延迟，记者表身份）+ 阿奇（另一半，替身→战友）。 [F120][F190]
- **中立/底层·狂草帮**：奥丁创建，收容两族孤儿/残疾/老人；以撒、黛利拉皆成员。半义警。 [F180]
- **被加害的无辜者**：被造"圣女"并被摘角膜的诺族小女孩；诺族电台台长（被黛利拉受指使所杀）；程聿怀父亲程亮冬；2000大屠杀中死去的布雷族平民、狂草帮成员。 [F104][F124][F172][F183]

## 二、加害链（莱诺家族 → 世界）

1. **夏尔·莱诺 → 黎梦鲸**：心理治疗中移情，凭家族势力强娶并囚禁黎梦鲸，致其心理扭曲。动机：错把患者依赖当爱情 + 家族强权。后果：生下瑞法（羌青瓷），瑞法被母逼迫严酷训练、从小"知道什么不是爱"。 [F111]
2. **莱诺家族 + 缪家 → 布雷诺**：1995建电台、2000建金狮医院，以"公益设施"为名挑动两族争抢/屠杀。动机：制造战争 → 卖军火 → 战争横财。后果：1995争抢、2000中旬种族大屠杀。 [F103][F105]
3. **莱诺家族 → 诺族小女孩**：指使程走柳造假新闻包装其为布雷族"圣女"（F132/F702），指使缪宏谟摘其眼角膜制造异象（F142/F720）。动机：跨族挑动仇恨、维持伪善叙事。 [F104]
4. **莱诺家族 → 程亮冬（程聿怀父）**：1994在布雷诺将其害死。后果：成为程聿怀复仇线导火索。 [F124]
5. **伯纳德·莱诺 → 程聿怀**：1995发现程聿怀查到家族真相，欲杀其灭口。被瑞法以"为家族做事+抹除程记忆"为代价交易劝止。 [F113]
6. **卢特·莱诺（娶程走柳）+ 伯纳德·莱诺（幕后动机）→ 程走柳**：羌青瓷叔叔卢特娶程走柳；其背后动机来自伯纳德——伯纳德一直知道程走柳是程聿怀的妹妹，为控制"延迟"而要把程的妹妹留在莱诺家族身边。（防杠第2条）动机：以亲属为人质控制延迟。 [F202]
7. **伯纳德·莱诺 → 黛利拉**：指使黛利拉杀诺族电台台长。 [F703][F716]
8. **瑞法·莱诺（羌青瓷）→ 以撒**：指使以撒刺杀莱诺家族成员（多为家主竞争对手）。动机：清除竞争者、爬上家主之位（以献芯片给程聿怀）。 [F114][F710]
9. **瑞法·莱诺 → 奥丁**：2000大屠杀后任奥丁心理医生，治疗中暗示"大屠杀是莱诺家族挑起的"，挑衅奥丁复仇。动机：复杂——既是莱诺代表，又暗中把矛头引向家族（为最终献祭芯片埋线）。后果：瑞法与奥丁彻底对立。 [F181]
10. **蒋伯驾（受瑞法提议）→ 金狮医院屠杀**：应瑞法提议雇佣兵在金狮医院展开屠杀，为双方家族谋战争横财。 [F707][F152]

## 三、复仇链（怒河/延迟/众人 → 莱诺家族）

1. **程走柳 + 缪宏谟 → 创建怒河**：大屠杀一年后找奥丁，提供军火钱财组建正规复仇组织"怒河"。动机：赎罪（曾被莱诺威逼害死无辜者）+ 复仇。隐居幕后。 [F130][F131][F182]
2. **奥丁 → 怒河表面指挥官**：同意出任，并暗中保护程走柳/缪宏谟（给她们"自然老死"的死法）。动机：不忍二人寻死、收容受苦之人。 [F182][F131]
3. **以撒、黛利拉 → 怒河杀手**：执行刺杀。以撒受瑞法指使杀莱诺成员；黛利拉受伯纳德指使杀电台台长（这些"加害"行为同时也是其"认罪"自污的素材）。 [F710][F716]
4. **程聿怀 + 阿奇 → "延迟"**：领导布雷族对抗诺族官方。程聿怀失忆后为查父死真相二次成为延迟。 [F120][F190]
5. **蒋伯驾 → 卧底揭露**：潜伏莱诺家族多年的卧底记者，最终自爆身份、带头抛证据。 [F150][F705]
6. **羌青瓷（瑞法）→ 献祭自己**：暗中助程聿怀等揭穿家族；让程聿怀取出自己心脏芯片（家族所有罪证），以死成全复仇。动机：对程聿怀十五年的爱。 [F114][F115][F102]
7. **2005纪念会清算**：奥丁+怒河守门牺牲（受控燃烧①）；众人当众自污认罪、抛罪证、放音频，曝光莱诺家族六大罪（受控燃烧②）。 [F106][F183][F201][F700–F720]

## 四、情感/血缘关系网

- **羌青瓷（瑞法·莱诺）↔ 程聿怀**：1990相爱→程被拒后追求成功→1994程父被害→1995程查真相→瑞法抹忆救程→2000重逢结婚(程失忆)→羌演"不爱"压抑真爱→2005程取芯片致羌死→程方知1990即相爱。纯爱（对外伪装纯恨）。 [F112–F115]
- **程聿怀 ↔ 程走柳**：亲兄妹（差5岁）。走柳假意吵架赶走兄长助其前程；走父澄清走新闻路。 [F133]
- **蒋伯驾 ↔ 缪宏谟**：同父异母兄妹（差2岁，缪家家主之子）。蒋把"小缪"照顾得好。 [F143]
- **蒋伯驾 ↔ 程走柳**：cp。走柳以陌生人身份给蒋过生日→蒋"活过来"→蒋一次次回到推开他的走柳身边。 [F153]
- **卢特·莱诺 ↔ 程走柳 / 蒋伯驾**：羌青瓷叔叔卢特（年近六十）娶程走柳，并收蒋伯驾为义子（因此场上奥丁起哄蒋伯驾应改口称程走柳"婶/母"）。注意：娶程走柳、收蒋伯驾义子均归卢特，**非**伯纳德。 [F202]
- **缪宏谟 ↔ 以撒**：cp。以撒因目睹缪被未婚夫戴戒指、缪痛苦表情而觉醒"爱是让对方自由"。"爱格塑造人格"。 [F162]
- **黛利拉 ↔ 阿奇**：cp。阿奇遇黛利拉重获生活；2003黛受怒河指派杀"延迟"=亲手杀阿奇；后幻想出"假阿奇"人格陪伴；结局假阿奇让黛恢复记忆、学会自爱。 [F173][F174][F192]
- **奥丁 ↔ 众人**：奥丁视黛利拉为女儿、以撒为儿子、程走柳/缪宏谟为朋友、想让蒋伯驾做家人、把程聿怀当同行者。狂草帮/怒河大家长。 [F180][F171][F161]
- **程聿怀 ↔ 阿奇**：君臣/战友/朋友，各为半个"延迟"。场上不互动（程隐藏延迟身份）。 [F120][F190]
- **缪宏谟 ↔ 黎挽南（母）**：母亲被迫嫁缪家、善良眼科医生，教缪"善待别人、救人"，缪钦佩认同母亲成为眼科医生。 [F140]
- **蒋伯驾 ↔ 黎挽南/肖须言/程走柳**：这些无血缘之人以爱教蒋"你不是工具，你是个人"。肖须言为记者、对蒋言传身教（也是蒋对生命谦逊的来源）。 [F151][F152]

## 五、动机—因果速记（一句话链）

- 莱诺家族**贪战争财** → 反复在布雷诺挑火（电台/医院）→ 害死无数无辜（含程父、圣女女孩、布雷族平民）→ 激起怒河复仇 → 2005清算倾覆。
- 夏尔**强娶黎梦鲸** → 瑞法畸形成长、不懂爱 → 被程聿怀治愈相爱 → 为护程而抹忆+献祭芯片 → 以死成全复仇。
- 程聿怀**失父+被抹忆** → 二次成为延迟查真相 → 一度"变得像莱诺"（伯纳德误判/真创伤）→ 终在羌助力下清算家族 → 羌死后方知相爱。
- 程走柳/缪宏谟**被莱诺威逼害无辜** → 罪孽感+复仇 → 创建怒河、欲杀凶手后自尽 → 被奥丁暗中保全。
- 黛利拉**战争之子缺爱** → 渴求爱 → 杀爱人阿奇成全其解脱 → 解离障碍幻想假阿奇 → 终学自爱。
- 阿奇**只能在死亡中显价值的孤独** → 为大义不杀程聿怀、为爱让黛利拉杀自己 → 死后以幻想人格守护黛利拉。
